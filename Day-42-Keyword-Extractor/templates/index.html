How It Works

Embedding Method

Uses Sentence Transformers to compute semantic embeddings, then applies MMR (Maximal Marginal Relevance) to select keywords that are both relevant and diverse.

Clustering Method

Groups similar keywords using K-Means clustering, then selects the most representative keyword from each cluster based on document relevance.

Tech Stack

Sentence Transformers (all-MiniLM-L6-v2), Scikit-learn for clustering, Flask backend, and TailwindCSS for the UI.
